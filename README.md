# holesail-gui
## About:
A GUI App for holesail, nothing much. For Holesail v2.
Kinda made by Google Gemini but it works!

## Instructions:
1. Download `install-gui.js` and put it somewhere.
2. Install Node.JS 22 LTS OR other versions above that version.
3. Navigate PowerShell, cmd, or Terminal to that directory where you put `install-gui.js`.
4. Run
```sh
node install-gui.js
```
5. Run as instructed in the output of the command.
6. Done! Enjoy!
